# 前后端服务启动说明

## 方式一：单独启动（推荐开发调试时使用）

### 1. 启动后端服务
```bash
# 进入后端目录
cd back-end
# 启动后端服务
python run.py
```

### 2. 启动前端服务
```bash
# 进入前端目录
cd front-end
# 启动前端服务
npm run dev
```

## 方式二：一键同时启动（推荐快速开发时使用）

```bash
# 进入前端目录
cd front-end
# 同时启动前后端服务
npm run start-all
```

## 访问说明

- 前端服务默认地址：http://localhost:8080
- 后端服务默认地址：http://localhost:5000

## 注意事项

1. 使用 `npm run start-all` 前，请确保：
   - 前端已安装依赖（运行过 `npm install`）
   - Python环境已安装必要依赖（可通过 `pip install -r back-end/requirements.txt` 安装）

2. 如果遇到端口冲突，请修改相应配置文件：
   - 前端端口：修改 `front-end/vue.config.js`
   - 后端端口：修改 `back-end/.env` 或 `run.py`

3. 停止服务时，按 `Ctrl+C` 即可同时终止前后端服务
