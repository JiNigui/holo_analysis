# 路由问题修复说明

## 问题分析

### 错误日志
```
127.0.0.1 - - [07/Dec/2025 21:47:25] "POST /api/upload/images/batch HTTP/1.1" 404 -
```

### 根本原因

**后端路由注册问题：重复的 `/upload` 前缀**

#### 后端路由结构
```python
# app/api/__init__.py
api_bp.register_blueprint(upload_bp, url_prefix='/upload')  # ← 第一个 /upload

# app/api/upload.py
@upload_bp.route('/upload/images/batch', methods=['POST'])  # ← 第二个 /upload（错误！）
```

**导致的完整路径：**
```
/api (主蓝图) + /upload (子蓝图前缀) + /upload/images/batch (路由)
= /api/upload/upload/images/batch  ← 错误的路径
```

**前端请求的路径：**
```
/api/upload/images/batch  ← 正确的期望路径
```

**结果：** 404 Not Found

---

## 修复方案

### 修改后端路由（去掉重复的 `/upload` 前缀）

**文件：** `back-end/app/api/upload.py`

#### 修改1：单文件上传API
```python
# 修改前（错误）
@upload_bp.route('/upload/image', methods=['POST'])

# 修改后（正确）
@upload_bp.route('/image', methods=['POST'])
```

**完整路径：** `/api/upload/image` ✅

---

#### 修改2：文件删除API
```python
# 修改前（错误）
@upload_bp.route('/upload/delete', methods=['POST'])

# 修改后（正确）
@upload_bp.route('/delete', methods=['POST'])
```

**完整路径：** `/api/upload/delete` ✅

---

#### 修改3：批量上传API（我们使用的）
```python
# 修改前（错误）
@upload_bp.route('/upload/images/batch', methods=['POST'])

# 修改后（正确）
@upload_bp.route('/images/batch', methods=['POST'])
```

**完整路径：** `/api/upload/images/batch` ✅

## 为什么会有这个问题？

### 代码设计问题

在Flask中，Blueprint的URL前缀会自动拼接到路由上：

```python
# 当你这样注册蓝图时：
api_bp.register_blueprint(upload_bp, url_prefix='/upload')

# 那么upload_bp中的路由会自动加上 /upload 前缀
# 所以路由应该写成：
@upload_bp.route('/images/batch')  # 正确 ✅

# 而不是：
@upload_bp.route('/upload/images/batch')  # 错误 ❌
# 这会导致路径变成 /api/upload/upload/images/batch
```

### 正确的思路

**蓝图前缀 + 路由路径 = 完整URL**

```python
# 示例1：用户API
api_bp.register_blueprint(user_bp, url_prefix='/user')
@user_bp.route('/login')  # 完整路径：/api/user/login ✅

# 示例2：项目API
api_bp.register_blueprint(project_bp, url_prefix='/project')
@project_bp.route('/')  # 完整路径：/api/project/ ✅

# 示例3：上传API（修复后）
api_bp.register_blueprint(upload_bp, url_prefix='/upload')
@upload_bp.route('/images/batch')  # 完整路径：/api/upload/images/batch ✅
```
